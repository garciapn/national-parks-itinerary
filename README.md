# The Great Escape 🏕️

An adventure-themed National Parks itinerary dashboard for Paolo & Brooke's 2026 road trips.

## Trips

| Trip | Park | Dates | Travel |
|------|------|-------|--------|
| 1 | Zion + Bryce Canyon | April 10-12 | 4.5hr drive from SD |
| 2 | Yosemite | May 8-10 | 6.5hr drive from SD |
| 3 | Grand Canyon | May 29-31 | 5hr drive from SD |
| 4 | Yellowstone + Grand Teton | July 10-14 | Fly SAN→JAC |

## Features

- **Live countdown** — Days, hours, minutes until your next trip
- **Day-by-day itineraries** — Full schedule for each park with times and activity details
- **NPS passport stamp aesthetic** — Each trip has its own color and stamp design
- **Weather tips** — What to expect and what to pack per trip
- **Interactive booking checklist** — Check off reservations as you make them (saved to localStorage)
- **Trip stats** — Miles from San Diego, days away, parks and states covered
- **America the Beautiful pass** — Notes on which entrance fees are covered

- Single `index.html` file — no build step, no framework, no backend
- Vanilla HTML + CSS + JavaScript
- Embedded SVG mountain silhouettes and topographic patterns
- Google Fonts (Playfair Display + Lato)
- localStorage for checkbox persistence
